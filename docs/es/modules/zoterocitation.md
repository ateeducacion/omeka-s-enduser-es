# Citas de Zotero El [módulo Zotero Citation](https://omeka.org/s/modules/ZoteroCitations/){target=_blank} permite a los usuarios acceder a las bibliotecas de Zotero para añadir citas utilizando el editor de texto enriquecido integrado.

Una vez activado, los usuarios configuran su acceso a Zotero desde sus ajustes de usuario. Se podrá acceder a sus bibliotecas Zotero a través de HTML y otros bloques en [páginas del sitio](../sites/site_pages.md). ## Ajustes de Zotero El módulo de citas Zotero añade una serie de opciones de configuración a la página de ajustes del usuario. Estos son los ajustes predeterminados del módulo.

- Estilo de cita: seleccione uno de los formatos de cita disponibles en el menú desplegable. - Configuración regional de la bibliografía: seleccione una configuración regional de idioma para mostrar la cita y la bibliografía. - Envoltura de enlaces de bibliografía: marque esta opción si desea que las URL de las bibliografías se envuelvan con el resto del texto de la entrada. - Tipo de biblioteca API: seleccione si desea acceder a una biblioteca de usuario o a una biblioteca de grupo.
- ID de la biblioteca API: introduzca el ID numérico de la biblioteca a la que desea acceder. - Clave API: si desea acceder a una biblioteca que no es pública, deberá generar una clave API desde la cuenta Zotero propietaria de la biblioteca. Introduzca esa clave aquí. - Ordenar búsqueda por: seleccione un campo de ordenación para los resultados de búsqueda al acceder a su biblioteca: título, creador o fecha de modificación. ![Opciones de citas de Zotero en la sección de configuración del usuario de la vista de edición de usuario; el estilo de citas está configurado en Chicago Manual of Style, la configuración regional de la bibliografía está configurada en inglés, la opción de ajuste de enlaces de la bibliografía está marcada, el tipo de biblioteca API está configurado en Usuario, el ID de la biblioteca API está configurado en 15, la clave API está en blanco y la orden de búsqueda está configurada en título](../modules/modulesfiles/zoterocitation_usersettings.png)

## Acceso a Zotero para citas y bibliografías Dentro de las páginas del sitio, se pueden añadir citas y bibliografías a cualquier elemento que tenga un editor de texto enriquecido, como el [bloque HTML](../sites/site_pages.md#html) o con leyendas para elementos. Cuando el módulo está activo, añade un icono «Z» rojo al editor de texto enriquecido. 

![Imagen de los comandos del editor WYSWIG con el icono Z rojo](../modules/modulesfiles/zoterocitation_redZ.png) Coloca el cursor en el lugar deseado de la ventana de edición y haz clic en el icono para abrir el cuadro de diálogo de selección de Zotero, que tiene tres pestañas: - **Añadir cita**: busca el elemento individual utilizando la entrada de texto. Haga clic en «Buscar en la biblioteca». Seleccione una entrada de los resultados. Solo se puede añadir una cita cada vez. Haga clic en «Aceptar» para generar la cita en el editor. - **Añadir bibliografía**: si ha añadido varias citas en el editor, puede colocar el cursor para generar una bibliografía. Marque la casilla de autorización y haga clic en el botón **Aceptar** para generar la bibliografía en el editor.
- **Configuración**: Aunque haya introducido la configuración predeterminada en su página de usuario, es posible que desee cambiar esas selecciones durante la creación de contenido. Aquí puede cambiar el estilo de cita, la configuración regional de la bibliografía, el tipo de biblioteca, el ID de la biblioteca, la clave API si utiliza una y el elemento de clasificación. Al ajustar la configuración, solo se modifican los parámetros para la generación actual de citas/bibliografía. Una vez que haya ajustado la configuración, vuelva a las pestañas «Añadir cita» o «Añadir bibliografía» para aplicar la configuración a las acciones que desee. ![Cuadro de diálogo de adición de citas de Zotero con la pestaña Añadir cita activa](../modules/modulesfiles/zoterocitation_addcitation.png)

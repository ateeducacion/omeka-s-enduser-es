# Editor CSS El [módulo Editor CSS](https://omeka.org/s/modules/CSSEditor){target=_blank} le permite escribir CSS desde la interfaz de administración de Omeka S.

Una vez activado en la sección [módulos](https://omeka.org/s/docs/user-manual/modules/){target=_blank} del panel de administración, el editor CSS se utiliza sitio por sitio. Si es nuevo en el uso de CSS, le recomendamos los siguientes recursos gratuitos para empezar:

* [«Aprender CSS» de Mozilla](https://developer.mozilla.org/en-US/docs/Web/CSS){target=_blank} * [Curso «Aprender CSS» de Codecademy](https://www.codecademy.com/learn/learn-css){target=_blank}
* [«Conceptos básicos de CSS» de Marksheet](https://marksheet.io/css-basics.html){target=_blank} La forma más fácil de aprender los componentes de las páginas de Omeka S para la edición de CSS es utilizar la herramienta Inspeccionar de tu navegador cuando veas el tema elegido.

## Aplicar CSS específico del sitio Si el editor CSS está activo, aparecerá un enlace al editor CSS en el menú contextual de cada sitio. ![Menú contextual del sitio Omeka S con la entrada del editor CSS como página activa.](modulesfiles/csseditor_contextmenu.jpg)

El primer área de texto grande es donde se escriben los estilos individuales. Utilice esa área de texto como si fuera un archivo de hoja de estilos. Esto cargará una línea en cada página pública del sitio Omeka S elegido, en el encabezado, que tendrá este aspecto: ` <link href="/yoursiteslug/css-editor" media="screen" rel="stylesheet" type="text/css"> `

Esta línea aparecerá después de las hojas de estilo que provienen de los valores predeterminados de Omeka y del tema que hayas elegido. Por lo tanto, las entradas aquí deberían anular otros estilos establecidos en esos archivos, a menos que se hayan marcado como `!important`. Puede que haya otros CSS personalizados cargándose en el encabezado debajo de esta línea, particularmente de configuraciones de temas como el color de acento principal o la altura del banner, que a su vez pueden anular tu CSS personalizado.

![Interfaz del módulo Editor CSS.](modulesfiles/csseditor_interface.jpg) El Editor CSS también le permite incluir hojas de estilo externas introduciendo sus URL. No hay límite en el número de URL de hojas de estilo externas que puede introducir. Cada entrada de texto puede contener una sola URL, y se pueden crear entradas adicionales haciendo clic en el botón «Añadir otra hoja de estilo».

Para eliminar hojas de estilo externas, borra las entradas de texto o haz clic en el icono de la papelera si hay varios campos de hojas de estilo. ![Campo de hoja de estilo externa dentro de la interfaz del módulo Editor CSS, botón de eliminación (icono de la papelera) resaltado.](modulesfiles/csseditor_remove.jpg) ## Tutorial: Uso de una fuente web de Google Puedes hacer que tu sitio Omeka S sea más distintivo utilizando una fuente personalizada. [Google ofrece una biblioteca gratuita de fuentes web](https://fonts.google.com/){target=_blank}, y este tutorial le mostrará cómo aplicar una fuente web a un sitio Omeka S utilizando el tema «Predeterminado» a través de la interfaz del editor CSS. A modo de referencia, el tema «Predeterminado» tiene este aspecto, utilizando la fuente «Open Sans».

![Sitio web Omeka S con el tema «Predeterminado». Todo el texto utiliza «Open Sans».](modulesfiles/csseditor_before.jpg) Este tutorial sustituirá «Open Sans» por la familia de fuentes «Lato». 1. Vaya a [Google Fonts](https://fonts.google.com/){target=_blank}. Busque la familia tipográfica «Lato» y haga clic en el botón naranja «+» para seleccionarla. ![Página principal de Google Fonts con la familia tipográfica «Lato» resaltada](modulesfiles/csseditor_tutorial1.jpg)<br> 2. Aparecerá una barra con la etiqueta «1 familia seleccionada» en la esquina inferior derecha. Haga clic en esta barra.
  ![Barra de la familia de fuentes seleccionada resaltada en la página de Google Fonts](modulesfiles/csseditor_tutorial2.jpg)<br> 3. La barra abrirá un panel con la información que necesitará para utilizar Lato en su sitio web. La primera sección, «Embed this font» (Incrustar esta fuente), contiene la URL de la hoja de estilos externa que necesita. Seleccione la URL en el atributo `href`, como se muestra en la imagen.
  ![Sección «Embed this font» con la URL de la hoja de estilos externa («https://fonts.googleapis.com/css?family=Lato&display=swap») resaltada.](modulesfiles/csseditor_tutorial3.jpg)<br> 4. Copie esta URL en uno de los campos «External stylesheets» (Hojas de estilos externas) del editor CSS.
  ![Campo de hoja de estilos externa dentro de la interfaz del módulo Editor CSS, con la URL de la hoja de estilos pegada.](modulesfiles/csseditor_tutorial4.jpg)<br> 5. De vuelta en el panel Google Fonts, hay una segunda sección titulada «Especificar en CSS». Copia la regla `font-family`.
  ![Sección «Especificar en CSS» con la regla font-family («font-family: 'Lato', sans-serif;») resaltada.](modulesfiles/csseditor_tutorial5.jpg)<br> 6. Para este tutorial, vas a establecer Lato como la fuente predeterminada del sitio. Para ello, en el área de texto grande «CSS» del editor CSS, establece la familia de fuentes del elemento `body` utilizando la regla que acabas de copiar. ![Área de texto CSS dentro de la interfaz del módulo del editor CSS, con la regla font-family del cuerpo pegada («body {font-family: "Lato", sans-serif;}»).](modulesfiles/csseditor_tutorial6.jpg)<br>
7. Haga clic en el botón «Guardar» situado en la esquina superior derecha. Ahora, el tema «Predeterminado» debería tener este aspecto. ![Sitio Omeka S con el tema «Predeterminado», todo el texto se muestra con la fuente «Lato»](modulesfiles/csseditor_after.jpg)

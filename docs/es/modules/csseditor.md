# Editor CSS El [módulo Editor CSS](https://omeka.org/s/modules/CSSEditor){target=_blank} le permite escribir CSS desde la interfaz de administración de Omeka S.

Una vez activado en la sección [módulos](https://omeka.org/s/docs/user-manual/modules/){target=_blank} del panel de administración, el editor CSS se utiliza sitio por sitio. Si es nuevo en el uso de CSS, le recomendamos los siguientes recursos gratuitos para empezar:

* [«Aprender CSS» de Mozilla](https://developer.mozilla.org/en-US/docs/Web/CSS){target=_blank} * [Curso «Aprender CSS» de Codecademy](https://www.codecademy.com/learn/learn-css){target=_blank}
* [«Conceptos básicos de CSS» de Marksheet](https://marksheet.io/css-basics.html){target=_blank} La forma más fácil de aprender los componentes de las páginas de Omeka S para la edición de CSS es utilizar la herramienta Inspeccionar de tu navegador cuando veas el tema elegido.

## Aplicar CSS específico del sitio Si el editor CSS está activo, aparecerá un enlace al editor CSS en el menú contextual de cada sitio. ![Menú contextual del sitio Omeka S con la entrada del editor CSS mostrada como página activa.](modulesfiles/csseditor_contextmenu.jpg) El primer área de texto grande es donde se escriben los estilos individuales. Utilice esa área de texto como si fuera un archivo de hoja de estilos. Esto cargará una línea en cada página pública del sitio Omeka S que hayas elegido, en el encabezado, que tendrá este aspecto: ` <link href="/yoursiteslug/css-editor" media="screen" rel="stylesheet" type="text/css"> ` Esta línea aparecerá después de las hojas de estilo que provienen de los valores predeterminados de Omeka y del tema que hayas elegido. Por lo tanto, las entradas aquí deberían anular otros estilos establecidos en esos archivos, a menos que se hayan marcado como `!important`. Puede haber otros CSS personalizados cargándose en el encabezado debajo de esta línea, particularmente de configuraciones de temas como el color de acento principal o la altura del banner, que pueden anular su CSS personalizado a su vez. ![Captura de pantalla de la interfaz del módulo CSS Editor.](modulesfiles/csseditor_interface.jpg)

El editor CSS también le permite incluir hojas de estilo externas introduciendo sus URL. No hay límite en el número de URL de hojas de estilo externas que puede introducir. Cada entrada de texto puede contener una sola URL, y se pueden crear entradas adicionales haciendo clic en el botón «Añadir otra hoja de estilo». Para eliminar hojas de estilo externas, borre las entradas de texto o haga clic en el icono de la papelera si hay varios campos de hojas de estilo.

![Captura de pantalla de la interfaz del módulo Editor CSS, recortada para centrarse en el campo de la hoja de estilo externa con el botón de eliminación (icono de la papelera) resaltado.](modulesfiles/csseditor_remove.jpg) ## Tutorial: Uso de una fuente web de Google Puede hacer que su sitio Omeka S sea más distintivo utilizando una fuente personalizada. [Google ofrece una biblioteca gratuita de fuentes web](https://fonts.google.com/){target=_blank}, y este tutorial le mostrará cómo aplicar una fuente web a un sitio Omeka S utilizando el tema «Predeterminado» a través de la interfaz del editor CSS. A modo de referencia, el tema «Predeterminado» tiene este aspecto, utilizando la fuente «Open Sans».

![Captura de pantalla del sitio Omeka S con el tema «Predeterminado». Todo el texto utiliza «Open Sans».](modulesfiles/csseditor_before.jpg) Este tutorial sustituirá «Open Sans» por la familia de fuentes «Lato». 1. Vaya a [Google Fonts](https://fonts.google.com/){target=_blank}. Busque la familia tipográfica «Lato» y haga clic en el botón naranja «+» para seleccionarla. ![Página principal de Google Fonts con la familia tipográfica «Lato» resaltada](modulesfiles/csseditor_tutorial1.jpg)<br> 2. Aparecerá una barra con la etiqueta «1 familia seleccionada» en la esquina inferior derecha. Haga clic en esta barra.
  ![Vista recortada de la página de Google Fonts con la barra de la familia de fuentes seleccionada resaltada](modulesfiles/csseditor_tutorial2.jpg)<br> 3. La barra abrirá un panel con la información que necesitará para utilizar Lato en su sitio web. La primera sección, «Embed this font» (Incrustar esta fuente), contiene la URL de la hoja de estilos externa que necesita. Seleccione la URL en el atributo `href`, como se muestra en la imagen.
  ![Captura de pantalla de la sección «Insertar esta fuente» con la URL de la hoja de estilos externa («https://fonts.googleapis.com/css?family=Lato&display=swap») resaltada.](modulesfiles/csseditor_tutorial3.jpg)<br> 4. Copie esta URL en una de las entradas «Hojas de estilos externas» del Editor CSS.
  ![Captura de pantalla de la interfaz del módulo Editor CSS, recortada para centrarse en el campo de la hoja de estilos externa con la URL de la hoja de estilos pegada.](modulesfiles/csseditor_tutorial4.jpg)<br> 5. De vuelta al panel Google Fonts, hay una segunda sección titulada «Especificar en CSS». Copia la regla `font-family`.
  ![Captura de pantalla de la sección «Especificar en CSS» con la regla font-family («font-family: 'Lato', sans-serif;») resaltada.](modulesfiles/csseditor_tutorial5.jpg)<br>
6. Para este tutorial, vas a establecer Lato como la fuente predeterminada del sitio. Para ello, en el área de texto grande «CSS» del Editor CSS, establece la familia de fuentes del elemento `body` utilizando la regla que acabas de copiar.
  ![Captura de pantalla de la interfaz del módulo Editor CSS, recortada para centrarse en el área de texto CSS con la regla font-family del cuerpo pegada ('body {font-family: "Lato", sans-serif;}').](modulesfiles/csseditor_tutorial6.jpg)<br> 7. Haga clic en el botón «Guardar» en la esquina superior derecha. Ahora el tema «Predeterminado» debería tener este aspecto.
![Captura de pantalla del sitio Omeka S con el tema «Predeterminado», donde todo el texto se muestra con la fuente «Lato»](modulesfiles/csseditor_after.jpg)

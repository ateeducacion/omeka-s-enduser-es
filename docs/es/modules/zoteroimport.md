# Importación de Zotero El [módulo de importación de Zotero](https://omeka.org/s/modules/ZoteroImport){target=_blank} le permite conectar una instancia de Omeka S a una biblioteca de Zotero e importar elementos de dicha biblioteca. ## Prepare su biblioteca de Zotero Para poder importar desde una biblioteca de Zotero, debe tener acceso API a la biblioteca. Para utilizar la importación de Zotero, necesitará el ID del grupo o del usuario: * Los ID individuales se encuentran en la pestaña Feeds/API de la configuración de Zotero; hay una frase que dice «Su ID de usuario para usar en la llamada API es». * Los ID de grupo se encuentran en la página de la biblioteca del grupo Zotero, mirando la URL de la biblioteca, que será `zotero.org/groups/<número>`. Si vas a `https://api.zotero.org/groups/<número>`, deberías obtener información en esa página. Es posible que también quieras tener el ID de una colección específica y una clave API para importar archivos. * Las claves de colección aparecen en la URL de una página de colección: son la secuencia alfanumérica al final de la URL, después de «collections/».
* Para generar una clave API, vaya a Configuración en su cuenta de Zotero, a la pestaña Feeds/API, y haga clic en el enlace *Crear nueva clave privada*. ## Importar datos Para importar, vaya a la sección Importación de Zotero en la navegación de la izquierda del panel de administración de su instalación de Omeka S. Esto le llevará automáticamente a la subsección Importación, también en la navegación de la izquierda.

![Opciones de importación de Zotero que se enumeran a continuación.](modulesfiles/zoteroimport_new.png) Para importar desde una biblioteca de Zotero: * Elija un conjunto de elementos al que se añadirán los elementos importados (obligatorio). * Elija entre bibliotecas de **Usuario** o **Grupo** con los botones de opción (obligatorio). 
* Introduce el ID de la biblioteca (es el número de ID del grupo o del usuario, como se ha explicado anteriormente). * Si importas una colección, introduce la clave de la colección. * Introduce tu clave API para importar datos y/o archivos privados (opcional para la importación, pero necesario para importar archivos). * Haz clic en la casilla de verificación para importar archivos. La clave API es necesaria para importar archivos. * Las instantáneas de páginas HTML se importarán como archivos zip. 
* La opción «Añadidos después de» le permite importar solo aquellos elementos de Zotero añadidos a la biblioteca después de la fecha y hora que elija. Puede utilizar esto como una forma de actualizar desde una importación anterior sin volver a añadir archivos que ya haya importado. Cuando haya completado el formulario, haga clic en «Enviar» en la esquina superior derecha de la ventana del navegador. !!! Nota ¿Sus trabajos se inician pero no se completan? Es posible que tenga que [configurar la ruta de PHP](../configuration.md#php-path) para que su sistema pueda realizar el proceso en segundo plano para crear los elementos. Nota sobre los archivos: Las reglas para importar un elemento son las mismas que para crear un elemento en Omeka, por lo que si su biblioteca Zotero tiene archivos que son de un tipo de medio no permitido, los elementos no se importarán.

La página se cargará en la subsección Importaciones anteriores con el mensaje «Importando desde Zotero». Para comprobar el estado de la importación, recargue la página o haga clic en la sección [Tareas del panel de administración](../admin/jobs.md). Gestión de importaciones anteriores --------------------------------- Para ver y gestionar importaciones anteriores, haz clic en la sección Importación de Zotero en el menú de navegación de la izquierda y, a continuación, haz clic en la subsección «Importaciones anteriores» que aparece cuando se expande el menú. La página Importaciones anteriores muestra una tabla con las importaciones anteriores, ya sean exitosas, fallidas por error o deshechas. La tabla tiene las siguientes columnas: - **Biblioteca Zotero**: muestra el nombre de la biblioteca del usuario o del grupo.
- El botón **Deshacer**: una flecha curvada en sentido antihorario. - **Elementos añadidos antes**: la fecha de la importación. - **Estado del trabajo**: estado actual del trabajo. Al hacer clic en el texto del estado de un trabajo, se le redirigirá a la página de ese trabajo.
- **Recuento de elementos**: el número de elementos añadidos en la importación. Para deshacer una importación, haga clic en el botón «Deshacer» situado a la izquierda de la marca de tiempo **Elementos añadidos anteriormente**. Se abrirá un cuadro de diálogo en la barra lateral en el que se le pedirá que confirme que desea deshacer la importación y eliminar todos los elementos importados. Haga clic en «Confirmar» para continuar. 

![Página de importaciones anteriores con la barra lateral abierta para confirmar la acción de deshacer](modulesfiles/zoteroimport_undo.png) Las importaciones anteriores que ya se hayan deshecho tendrán un estado de trabajo de «Deshacer importación completada».
